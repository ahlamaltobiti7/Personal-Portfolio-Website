#  Personal Portfolio Website

##  Project Overview

This project is a personal portfolio website developed using **HTML** and **CSS**. It presents my personal information, academic background, technical skills, and contact details through a clean, responsive, and modern interface.

The website was created to practice front-end web development fundamentals and was successfully deployed online using **InfinityFree**.

---

##  Objectives

- Build a personal portfolio website using HTML and CSS.
- Practice front-end web development skills.
- Create a clean and responsive user interface.
- Showcase personal information and technical skills professionally.
- Publish the website online using InfinityFree.

##  Development & Deployment

This project was completed through the following steps:

1. Planned the structure and content of the portfolio website.
2. Learned and reviewed HTML and CSS concepts using **W3Schools**.
3. Developed the website using **Visual Studio Code**.
4. Tested the website locally in a web browser.
5. Created an **InfinityFree** hosting account.
6. Uploaded all project files to the **htdocs** directory.
7. Published the website and verified that it was working successfully online.
